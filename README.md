SPECYFIKACJA:
min. 8 znaków, max. 20 znaków
wymagane: mała litera, duża litera, liczba, znak specjalny

moc hasła:
Przyznaj punkt za:
1pkt - powyżej 12 znaków
1pkt - powyżej 2 znaki specjalne
1pkt - powyżej 2 liczby

Bazując na punktach wypisz jakie jest hasło:
0pkt - bardzo słabe
1pkt - hasło słabe
2pkt - hasło średnie
3pkt - hasło silne

UI:
Zrób ładną stronę, z polem do wpisywania hasła z wymaganiami, które po zatwierdzeniu wyświetla pasek z siłą hasła ( musi hasło być sprawdzane pod kątem wymagań )
